# Project Theme

Travel & Tourism Landing Page
A modern, clean, and mobile-friendly website showcasing destinations, tour packages, testimonials, a newsletter signup, and a contact section.

# Sections Implemented
1. Navbar (Sticky Header)
Sticky top navigation with logo & links
Responsive behavior: links hide on mobile
2. Hero Banner
Full-width background image (gradient overlay)
Animated fade-in text and CTA button
3. Popular Destinations (CSS Grid)
Auto-fit responsive grid
Image cards with hover-friendly design
4. Tour Packages Section
Flexbox-based cards
Displays duration, price, and CTA
5. How It Works
step process layout using Flexbox
Clean card structure
6. Testimonials
Avatar-based customer reviews
Responsive two-column layout
7. Blog Previews (Grid Layout)
Grid cards featuring travel blogs
Image previews + metadata
8. Stats Counter
Full-width dark section
Flexbox stats row
9. Newsletter Signup
Inline form
Converts to stacked form on mobile
10. Contact Section
Contact form + map mock
Two-column layout using Flexbox
